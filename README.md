# Flutter Novel Reader App 📚

A mobile novel reader application built with Flutter that allows users to import, organize, and read digital novels directly from their devices. The application supports EPUB and other document formats while providing a personal library system, reading progress tracking, bookmarks, and customizable reading settings.

## 📖 Overview

The Flutter Novel Reader App is designed to provide users with a convenient way to manage and read their personal collection of digital novels. Users can import novel files, automatically extract book information, store them in a library, and continue reading from where they stopped.

The application focuses on offline reading, organization, and a customizable reading experience.

---

# ✨ Features

## 📂 Novel Import System
- Import novels from device storage
- Support for EPUB and other reading formats
- Extract book metadata:
  - Title
  - Author
  - Cover image
  - Chapters

## 📚 Personal Library
- Add imported novels to the library
- View book covers and information
- Search and filter novels
- Organize books by categories
- Track recently opened books

## 📖 Reading System
- Chapter-based reading
- Page navigation
- Reading progress tracking
- Continue reading from the last position
- Custom reading settings

## 🔖 Bookmark System
- Save important chapters or pages
- Add reading notes
- Quickly return to saved sections

## 🎨 Reading Customization
- Light mode
- Dark mode
- Sepia mode
- Adjustable font size
- Adjustable text spacing

## 📊 Reading Statistics
- Reading progress percentage
- Recently read novels
- Reading history
